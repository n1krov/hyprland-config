## Estructura sugerida para la carpeta `hypr/`

```
hypr/
├── config.conf           # Configuración principal: imports y opciones globales
├── autostart.conf        # Aplicaciones y scripts que se ejecutan al inicio
├── keybindings.conf      # Atajos de teclado personalizados
├── workspace.conf        # Reglas y configuración de workspaces
├── appearance.conf       # Temas, barras, fonts, colores, etc.
├── windowrules.conf      # Reglas específicas para ventanas y aplicaciones
├── input.conf            # Configuración de mouse, teclado, touchpad
├── monitor.conf          # Layout y settings de monitores
├── animations.conf       # Configuración de animaciones y efectos
├── env.conf              # Variables de entorno y paths
├── scripts/
│   ├── autostart.sh      # Script de autostart
│   └── custom.sh         # Otros scripts personalizados
├── README.md             # Explicación de toda la configuración
```
